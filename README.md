# Alex .sh — Photography Portfolio

A dark cinematic photo portfolio showcasing street and travel photography from Moscow and beyond.

## Highlights
- Single-page site: Home, Gallery, About, Contact
- Responsive gallery with lazy-loaded images
- Dark cinematic theme with serif + sans typography
- GitHub Pages-ready static build

## Tech
- HTML
- CSS
- JavaScript (vanilla)

## Run locally
Open `index.html` in a browser (no build step required).
